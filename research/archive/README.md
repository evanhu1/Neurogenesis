# Archive

This archive contains completed experiment records, negative results, aborted
runs, conceptual reports, and the historical visual atlas. It is decision
history, not a statement that every historical mechanism still exists in the
current engine.

- `experiments/` contains empirical records.
- `reports/` contains adversarial audits and proposed mechanisms.
- `atlas.html` is the historical interactive report.
- `redesign.md` and `legacy-operating-procedure.md` are superseded designs kept
  for provenance.

Start from [`../INDEX.md`](../INDEX.md), which summarizes what was tested and
which mechanisms were rejected.
